# My Portfolio

This is my portfolio! It's very minimal right now, but in the future I plan to add more
content.

## Commands

| Command           | Action                                       |
| :---------------- | :------------------------------------------- |
| `npm install`     | Installs dependencies                        |
| `npm run dev`     | Starts a local dev server                    |
| `npm run build`   | Builds the website to  `./dist/`             |
